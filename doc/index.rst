=================

This is the documentation of **trixi** (Training & Retrospective Insights eXperiment Infrastructure).
**trixi** is a python package aiming to facilitate the setup, visualization and comparison of reproducible experiments,
currently with a focus on experiments using PyTorch.

You can jump right into the package by looking into our :ref:`quick-start-label`.
